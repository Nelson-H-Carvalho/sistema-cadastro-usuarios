# User Registration System

## Overview
This repository contains a user registration system designed to facilitate the creation, validation, and management of user accounts.

## Features
- User registration with email validation
- Password encryption for secure storage
- User account management
- Easy integration with other systems

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Nelson-H-Carvalho/sistema-cadastro-usuarios.git
   ```
2. Install the required dependencies:
   ```bash
   cd sistema-cadastro-usuarios
   npm install
   ```

## Usage
To start the user registration system, run:
```bash
npm start
```

## API Documentation
- **POST /register**: Registers a new user.
  - Body parameters:
    - `email`: User's email (required)
    - `password`: User's password (required)

- **GET /users**: Retrieves a list of registered users.
